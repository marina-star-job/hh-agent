3. Саммаризация документов
Выделение ключевой информации из большого объема текстов, без потери смысла
Решение:
1)	Текст передаем в сервис парсинга
2)	Далее предобработка данных, выделение глав и пр. 
3)	Применение экстрактивных методов – выбираем самые важные предложения
4)	Абстрактивные методы с применением LLM – сохраняют смысл, поддерживают естественность изложения. 
Получили баланс между точностью и естественностью.

4.	Сравнение документов
Поиск противоречий в связанных между собой документах. Например, в документе есть ссылки на ВНД. Не противоречат ли контексты документов друг другу. 
Решение: 
1) Для каждого абзаца со ссылкой — RAG достаёт топ-5 релевантных чанков из ВНД
2) LLM сравнивает контексты и выявляет смысловые противоречия

Сложный кейс: модель на первых тестах выдавала слишком много «ложных тревог» — находила несоответствия там, где их не было (разный стиль изложения, синонимы). Бизнес был недоволен: «проверять каждый флаг вручную — это ещё больше работы». 
Решение: добавили confidence-score и порог отсечения. Теперь модель показывает только расхождения с уверенностью > 0.75 — количество ложных срабатываний снизилось на 60%, удовлетворённость бизнеса выросла.
